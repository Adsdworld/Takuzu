◆ CalculateAngle()

void CalculateAngle ( void )

Calculates the angle of the joystick from the calibrated center position.

This function computes the angle between the joystick's current position and the calibrated center using the atan2 function. The resulting angle is expressed in degrees. The calculation is based on the difference between the current joystick X and Y positions and their respective calibration values.

Note
The angle is calculated using the formula: angle = atan2(dy, dx) * (180.0 / M_PI)
Returns
void

◆ Calibrate()

uint8_t Calibrate ( void )

Calibrates the joystick's neutral position.

This function performs a calibration of the joystick by sampling its X and Y values multiple times, averaging them to determine the neutral (centered) position. This is useful to offset hardware drift or joystick imperfections.

Visual feedback is given through a WS2812 LED matrix, which turns red during calibration and off afterward. Calibration status is logged at each step using printf.

The ADC readings are gathered using the UpdateJoystickX() and UpdateJoystickY() functions.

Returns
JOYSTICK_SUCCESS on success or JOYSTICK_FAILURE on failure.

void DetectButtonAndExecuteCallback ( void )

Detects joystick button presses and executes the associated callback.

This function checks the state of the joystick button, manages debouncing, and triggers a callback function when a button press is detected. It distinguishes between short and long presses based on predefined delays.

◆ InitJoystick()

uint8_t InitJoystick ( void )

Initializes the joystick hardware.

This function configures the GPIO pins used by the joystick:

  • The joystick button (digital input with pull-up)
  • The X and Y axes (analog inputs without pull-up/pull-down)

It also initializes the ADC module required for reading analog joystick values, and performs a calibration routine.

Returns
JOYSTICK_SUCCESS on success or JOYSTICK_FAILURE on failure.
